## 📌 Project Overview

This project is part of a structured AI/ML internship program focused on developing strong foundations in Python, data analysis, statistics, and machine learning readiness.

The dataset used contains **1000 student records**, including demographic information and academic performance scores in three subjects:
- Math
- Reading
- Writing

The main objective is to perform **Exploratory Data Analysis (EDA)**, statistical computations, feature engineering, and visualization to extract meaningful insights and prepare the data for machine learning tasks.

- Set up a Python data science environment
- Perform data loading and inspection
- Analyze dataset structure and quality

- Conduct statistical analysis
- Explore categorical and numerical distributions

- Analyze correlations between variables
- Engineer meaningful new features
- Build reusable Python functions
- Perform NumPy-based manual computations

## 🧪 Key Steps Performed

### 1. Environment Setup
- Installed and verified libraries: NumPy, Pandas, Matplotlib, Seaborn
- Ensured reproducible data analysis environment

### 2. Data Exploration
- Loaded dataset using Pandas
- Displayed head, tail, and random samples
- Checked data types, shape, and structure

### 3. Data Quality Check
- Identified missing values
- Checked duplicate entries
- Confirmed dataset cleanliness

### 4. Statistical Analysis
- Mean, median, mode, variance, and standard deviation
- Percentile analysis
- Z-score analysis for outlier detection

### 5. Data Visualization
- Histograms, box plots, scatter plots
- Heatmaps for correlation analysis
- Pair plots for multi-variable relationships
- Bar charts and pie charts for categorical insights

### 6. Feature Engineering
New features were created:
- Total Score
- Average Score
- Performance Grade (A–F)
- High Achiever Flag
- Lowest Performing Subject

### 7. Group Analysis
- Comparison of scores across gender, race, parental education, lunch type, and test preparation

### 8. NumPy Analysis
- Manual computation of statistics
- Z-score calculations
- Percentiles
- Sorting and indexing for top performers
- Pass/fail conversion

### 9. Advanced Visualization Dashboard
A professional 6-chart dashboard was created including:
- Score distributions
- Box plots
- Scatter plots with regression
- Grouped bar charts
- Pie charts of grade distribution

---

## 📊 Key Insights

- Reading and writing scores are highly correlated
- Math shows higher variability compared to other subjects
- Test preparation significantly improves student performance
- Parental education has a mild but noticeable effect on academic success
- Females perform better in reading and writing, while math shows smaller gender gaps
- Most students fall in grade categories B and C
